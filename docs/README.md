# GitHub Pages Hosting

This `docs` folder is a self-contained static presentation for the keynote:

- `index.html`
- `styles.css`
- `app.js`

## How to host on GitHub

1. Put the `madamlyn` folder into a GitHub repository.
2. In the repository settings, enable **GitHub Pages**.
3. Set the Pages source to:
   - Branch: `main` (or your default branch)
   - Folder: `/docs`
4. Save the settings.

GitHub Pages will publish the presentation from:

`https://<your-username>.github.io/<repo-name>/`

## Notes

- The presentation already links to the local research files in `../sources` and keynote docs in `../notes`.
- No build tool is required.
- Keyboard shortcuts:
  - `←` / `→` or `↑` / `↓`: move between slides
  - `S`: open sources drawer
  - `N`: toggle script mode
  - `O`: open overview
  - `Esc`: close open panels
